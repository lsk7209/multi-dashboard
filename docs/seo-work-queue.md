## 진짜 레버 (순위 상승 — 사이트별·레포별 작업)
순위 5~15위 페이지를 1페이지 상단으로 올리는 작업:
1. 상위 노출 페이지 콘텐츠 보강 (깊이, FAQ, 비교표, 1차 데이터)
2. 내부링크 강화 (허브→인기 페이지, 관련 페이지 상호링크)
3. 신선도 — tennisfrens 선수 페이지: 최신 경기/랭킹/시즌 업데이트
4. E-E-A-T·스키마 보강
5. 시간: 신규/개편 사이트는 색인 성숙 대기 (fastjob 등)
